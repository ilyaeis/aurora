# Aurora — AI-Driven 2D MMORPG

## Project Summary

Open-world 2D MMORPG where **all content is generated by small open-source AI models** — world, NPCs, quests, abilities, items, enemies. No hand-crafted content. Co-op for 2-10 players.

## Tech Stack

| Layer | Tech |
|-------|------|
| Frontend | React + TypeScript + PixiJS (Vite) |
| Backend | Python FastAPI + uvicorn |
| Database | PostgreSQL 16 + Redis 7 |
| Real-time | WebSockets |
| AI | Ollama (local, CPU) — Qwen2.5-3B primary |
| Containers | Docker Compose |

## Hardware Constraint

Snapdragon X Elite ARM laptop — **no CUDA, CPU inference only**.
- Use models ≤3B params, Q4/Q5 quantized
- Keep prompts short (≤200 tokens in, ≤300 out)
- Always use JSON mode for structured output
- Cache all AI output in Redis + PostgreSQL — never regenerate

## Running the Project

```bash
# Start backend services (DB + Redis + API)
docker compose up -d

# Start frontend dev server
cd frontend && npm run dev
```

- Frontend: http://localhost:5173
- Backend API: http://localhost:8000
- API docs: http://localhost:8000/docs
